# Hermes adapter

Hermes should read repository rules from `AGENTS.md` and portable workflow contracts from `.agents/`.

- Validation: `.agents/validation.md`
- Skill routes: `.agents/skill-routing.md`
- Current state: `STATE.md`
- Continuation: `.hermes/handoff.md`
- Navigation: `.hermes/code_landmarks.md`

This adapter contains no independent policy and must not drift from the portable source.
