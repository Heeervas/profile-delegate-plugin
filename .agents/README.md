# Portable agent contract

`.agents/` is the runtime-neutral operating layer for Profile Delegate.

- `validation.md` defines the canonical evidence required before release.
- `skill-routing.md` maps project work to installed workflow skills without copying their contents.

Runtime-specific adapters must point back here rather than duplicating policy. Repository authority remains `AGENTS.md`, `BRIEF.md`, `STATE.md`, accepted plans, and decision records.
